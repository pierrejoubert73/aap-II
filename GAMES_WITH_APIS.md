# Online Games with Extensive Public APIs

This document provides a curated list of online games and gaming platforms that offer extensive public APIs for developers. These APIs allow you to access game data, player statistics, match history, and more to build tools, websites, and applications.

## 1. Riot Games (League of Legends, Valorant, TFT, etc.)
Riot Games offers a comprehensive developer portal covering multiple titles including League of Legends, Teamfight Tactics, Legends of Runeterra, and Valorant.
*   **Capabilities:** Access match history, ranked statistics, champion mastery, summoner details, and server status.
*   **Documentation:** [Riot Developer Portal](https://developer.riotgames.com/)

## 2. Blizzard Entertainment (World of Warcraft, Diablo, StarCraft, etc.)
The Battle.net API provides access to data for Blizzard's major franchises.
*   **Capabilities:** Retrieve WoW character profiles, guild information, auction house data, Diablo leaderboards, and StarCraft II profile data.
*   **Documentation:** [Blizzard Battle.net Developer Portal](https://develop.battle.net/documentation)

## 3. Destiny 2 (Bungie)
Bungie provides a very powerful API for Destiny 2, which is used by popular item manager apps.
*   **Capabilities:** Read inventory, vault contents, character stats, progression, vendors, and milestones. It also allows some actions like equipping items and transferring gear.
*   **Documentation:** [Bungie.net API](https://github.com/Bungie-net/api)

## 4. EVE Online
EVE Online is known for its massive player-driven economy and has one of the most extensive APIs in gaming, known as ESI (EVE Swagger Interface).
*   **Capabilities:** Access character sheets, corporation data, market orders, assets, wallet transactions, industry jobs, and mail.
*   **Documentation:** [EVE Swagger Interface (ESI)](https://esi.evetech.net/ui/)

## 5. Guild Wars 2
ArenaNet provides a robust, HTTP-based API for Guild Wars 2.
*   **Capabilities:** detailed account information, character equipment, inventory, achievements, trading post prices, and WvW (World vs World) match statuses.
*   **Documentation:** [Guild Wars 2 API Wiki](https://wiki.guildwars2.com/wiki/API:Main)

## 6. PUBG (PlayerUnknown's Battlegrounds)
The official PUBG API allows developers to access post-match data.
*   **Capabilities:** Retrieve match statistics, telemetry data (positions, kills, damage), player lifetime stats, and season rankings.
*   **Documentation:** [PUBG Developer Portal](https://developer.pubg.com/)

## 7. Dota 2 (via Steam Web API & OpenDota)
While Valve exposes data via the Steam Web API, the community-driven OpenDota API provides parsed replay data which is often more useful.
*   **Capabilities:** Match history, detailed replay analysis (skill builds, item timings, combat logs), and player MMR estimates.
*   **Documentation:** [Steam Web API](https://wiki.teamfortress.com/wiki/WebAPI) | [OpenDota API](https://docs.opendota.com/)

## 8. Magic: The Gathering (Scryfall)
While not a video game in the traditional sense (though MTG Arena exists), Scryfall provides the definitive API for MTG card data.
*   **Capabilities:** Search and retrieve card details, images, rulings, set information, and pricing data.
*   **Documentation:** [Scryfall API](https://scryfall.com/docs/api)

## 9. Hypixel (Minecraft Server)
Hypixel, one of the largest Minecraft server networks, offers a public API for its minigames.
*   **Capabilities:** Player online status, stats for specific games (SkyBlock, Bed Wars, etc.), guild information, and auction house data.
*   **Documentation:** [Hypixel Public API](https://api.hypixel.net/)

## 10. Clash of Clans
Supercell offers an official API for Clash of Clans.
*   **Capabilities:** Clan search, detailed clan information, war logs, league group information, and player labels.
*   **Documentation:** [Clash of Clans API](https://developer.clashofclans.com/)

## 11. Chess.com & Lichess
Both major online chess platforms offer excellent APIs.
*   **Chess.com:** Retrieve player profiles, game archives, and club data. [Documentation](https://www.chess.com/news/view/published-data-api)
*   **Lichess:** Open source and very developer-friendly. Stream current games, fetch user data, puzzle history, and analyze games. [Documentation](https://lichess.org/api)

## 12. Warframe
Digital Extremes exposes world state data, and the community has built excellent wrappers.
*   **Capabilities:** Current world state including alerts, invasions, void fissures, sortie information, and Baro Ki'Teer inventory.
*   **Documentation:** [Warframe Worldstate API (Unofficial Docs)](https://github.com/WFCD/warframe-worldstate-parser)

## 13. Elite Dangerous
Data is primarily accessed via player journal files, but services like EDSM (Elite Dangerous Star Map) and Inara have APIs aggregating this data.
*   **Capabilities:** Star system coordinates, body information, station market data, and flight logs.
*   **Documentation:** [EDSM API](https://www.edsm.net/en/api)
